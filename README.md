# 部署

> 采用`jekyll`搭建部署，https://jekyllrb.com

```bash
# install jekyll
gem install bundler jekyll

# 本地开发
jekyll serve
# Now browse to http://localhost:4000

# 生成静态文件到dist
# https://www.jekyll.com.cn/docs/usage/
jekyll build --destination dist 
```
