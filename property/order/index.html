#order

定义元素的排序。

默认属性 order: 0;

元素的排序按照其 HTML 代码中的排序。

One
order: 0

Two
order: 0

Target
order: 0

Four
order: 0

order: 1;

元素的排序与其同级元素具有相关性。最终的排序将会考虑到所有元素的排序值。

One
order: 0

Two
order: 0

Target
order: 1

Four
order: 0

order: -1;

值你可以使用负数。

One
order: 0

Two
order: 0

Target
order: -1

Four
order: 0

order: 9;

你可以为每一个元素设置不同的值。

One
order: 13

Two
order: -7

Target
order: 9

Four
order: 5
